# Web-Programming
網頁程式設計專題~

## 專案執行方式
請先進入專案主要資料夾：
```bash
cd Web-Programming/線上成功版/Website-main/Web-Programming-main/stamp-studio-online

安裝套件：

npm install

執行本機測試：

npm run dev

修改完成後，先建立正式版：

npm run build

再預覽正式版是否正常：

npm run preview

如果確認沒有問題，即可重新部署到 Firebase Hosting：

firebase deploy --only hosting
