請在 src/App.jsx 做這 3 個修改：

1. 在 import 區加這行：

import ScrapbookPlanner from "./ScrapbookPlanner.jsx";

建議放在 BoothChoose 那行附近：

import BoothChoose from "./BoothChoose.jsx";
import ScrapbookPlanner from "./ScrapbookPlanner.jsx";
import "./App.css";


2. 在 tab-switcher 裡加一顆按鈕：

<button
  className={activeTab === "scrapbook" ? "active" : ""}
  onClick={() => setActiveTab("scrapbook")}
>
  Scrapbook Planner
</button>

可以放在 Community Wall 按鈕後面。


3. 在 App.jsx 下方分頁顯示區加這行：

{activeTab === "scrapbook" && <ScrapbookPlanner currentUser={currentUser} />}

可以放在這行後面：

{activeTab === "community" && <CommunityWallPage refreshKey={refreshKey} currentUser={currentUser} />}


完成後執行：

npm run dev
